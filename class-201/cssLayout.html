MORE CSS LAYOUT

CSS treats each element as if it were in its own box.
This box will either be block level or an inline box
block level boxes start a new line and acts as the main building blocks of any layout
Inline boxes flow between surrounding text.

If one block level element sits inside another block-level element
then the outer box is known as the containing or parent element.

CSS has 3 positioning properties that allow you to control the layout of the page
normal flow, relative positioning, and absolute positioning.
You can specify the positioning scheme using the position property in css.
