HTML IMAGES

Images can be used to set the tone of a webpage.
If you are buliding from scratch, you want to find images that
convey your intended tone. You would want all these images stored in
the same folder just for good practice.
We use the image tag " " to add images. This is an empty element
meaning there is no corresponding closing tag. We use the source attribue " src="" "
to identify the path of an image. Images can have a set width and height or you can change
them to accomodate your own purposes
