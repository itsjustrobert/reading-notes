HTML TABLES

When representong a table, think in terms of a grid made up of rows and columns.
A bit like a spreadsheet

  • Tables represent information in grid format
    • each block in the grid is called a table cell. in html a table i written out row by row
  • the table element is used to create a table
  • the tr element is used to start a table row
  • each cell of a table is represented by the td element
  • The th element is used to represent the table heading
