HTML LINKS

Links alow a person to move from one webpage to another.
This is where we get the phrase surfing the web from.
There are a few common types that we will often run across

  • LINKS FORM ONE WEBPAGE TO ANOTHER
  • LINKS FROM ONE PAGE TO ANOTHER ON THE SAME SITE
  • LINKS FROM ONE PART TO ANOTHER PART OF A PAGE
  • LINKS THAT OPEN IN A NEW BROWSER OR WINDOW
  • LINKS THAT START YOUR EMAIL

LAYOUTS

Block level boxes start on a new line and act as the main building blocks of any layout
It is common to group a number of elements together inside a div tag.
That way you are able to control the flow of multiple elements.

    KEY CONCEPTS IN POSITIONING
  • INLINE ELEMENT flow in between surrounding text
  • BLOCK LEVEL ELEMENTS start a new line
  • FIXED POSITION keeps an element in one spot and you cannot apply position properties to it
